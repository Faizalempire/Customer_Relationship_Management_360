##### \# CRM360 - Customer Relationship Management System

##### 

##### CRM360 is a full-stack Customer Relationship Management platform designed to help organizations manage customers, leads, tasks, follow-ups, and business activities efficiently.

##### 

##### The system provides role-based access control for Admins, Sales Managers, and Sales Executives with a modern dashboard interface and secure backend APIs.

🔔 Notification System

CRM360 includes an integrated notification system that automatically generates notifications when:

A new Lead is assigned
A Customer is assigned
A Task is assigned
A Follow-up is scheduled

Notifications are stored in MongoDB and displayed to the assigned user through the Notifications module.
